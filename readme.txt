== Description ==

The "Simple MailChimp" WordPress plugin will make it very easy for you to add a simple, customizable MailChimp form to any page using shortcodes. Quick and easy!

== Installation ==

1. Upload the folder `SimpleMailChimp` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Follow the instructions on the settings page for the plugin
